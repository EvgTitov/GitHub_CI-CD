О проекте

Этот сайт посвящен изучению GitHub Actions и реализации CI/CD процессов.

План реализации:

  • Создание локального репозитория для хранения проекта

Цель проекта - освоить базовые принципы работы с GitHub Actions и настроить автоматическую сборку и деплой проекта.
