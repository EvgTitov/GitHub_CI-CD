Что такое CI/CD?

CI/CD (Continuous Integration/Continuous Delivery) - это методология разработки программного обеспечения, которая позволяет автоматизировать процесс интеграции кода и доставки приложений.

Настройка GitHub Actions

Создание рабочего процесса

Создайте файл .github/workflows/main.yml:

name: CI/CD Pipeline

on: [push]

jobs:
  build:
    runs-on: ubuntu-latest
    
    steps:
    - uses: actions/checkout@v2
    - name: Run a one-line script
      run: echo Hello, world!

Основные компоненты рабочего процесса

Триггеры

Определяют, когда запускается рабочий процесс:

on:
  push:
    branches: [ main ]
  pull_request:
    branches: [ main ]
